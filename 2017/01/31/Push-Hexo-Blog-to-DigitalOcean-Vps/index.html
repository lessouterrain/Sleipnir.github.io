迁移Blog至Digital Ocean VPS （准备篇）

故事的开始

首先会起意弄个VPS其实是因为想给博客换HTTPS，但CloudFlare总还是觉得不放心，便干脆租了个VPS。

VPS的选择

这个其实没那么复杂，大的VPS服务商也就那么几家，因为Github的学生包会送$50 Digital Ocean的优惠，加上对于一个静态博客来说，最便宜的$5/Month 的就足够用了。于是便开始着手折腾。如果你不是学生没法申请 Student Pack，我这边也有个$10的优惠码，虽然是一点点的优惠，但对$5/Month的包姑且也能试用2月的服务。

申请Github学生包

这个对于学生党来说相当的实惠。直接google Github student pack就能看到页面。首先必须的是教育机构的邮箱。通常来说多为.edu 结尾，日本不少大学是.ac.jp，德国大学邮箱后缀多为.de，等等。这个太大关系，一般来说正规高等教育机构的邮箱都很好用。按照网页填好姓名，邮箱，大学名称。然后是最后一项理由。这个我填的时候稍稍纠结了一下，因为实在是编不出很多冠冕堂皇的话来，就只填了两句。本来查教程的时候看到大约是几个小时到几天不等，甚至有不少被拒的情况，还有些担心理由不充分不给过。结果不到20分钟Github就发来了邮件告诉我申请成功，确实是意料之外。个人觉得，基本只要能确认是学生身份就会给的。
发来的邮件里第二条链接就是获取我们所要的 Student Pack，点击进入后能发现一大串各类服务的优惠。往下拉找到Digital Oceand 链接，点击然后记得复制学生优惠的激活码，并点击下方的链接跳转至 Digital Ocean 的网页。

Digital Ocean VPS的购买

跳转到的主页上就有个注册用的填写框，下面有个 Create Account。

注意！！
这边根据JFamily的博文里的提示，不要点右上角的 Sign up，那样你的优惠码就没有使用的机会了。

接着选好支付方式，Digital Ocean需要扣除一个月的费用作为押金。支持的支付方式为信用卡或PayPal。是的，没有支付宝。请选择一个对你来说相对方便的吧。作为留学党虽然两个都有，这里我还是选择了PayPal。在德国扯到银行总会是个相对麻烦的事。
填好优惠码付好款就可以创建vps系统了。

VPS内的操作系统安装

操作系统的选择

Digital Ocean在新建时提供操作系统预装服务，皆为Linux系，如Ubuntu，Fedora，Debian，CentOS等多种选择。

  • Fedora是这些系统里相对来说比较激进的发行版，更新较快。
  • Debian 和 CentOS 以稳定称著，多被用与服务器OS
  • Ubuntu 是基于 Debian 的商业驱动的发行版

选一个最熟悉用得最顺手的发行版吧。

SSH远程登录VPS

接着是SSH的远程登录的设定。你会看到一个“Add SSH Key”的蓝色按钮。虽然之后也能导入公匙，但这边建议在一开始就先导入。如果没有设定SSH 公匙，Digital Ocean 将会用邮件发送给你一个用于登录的用户名和密码。
点开后出现一个窗口需要填写public key的名称和内容，先放置在这边。

本地生成SSH Keys

各Linux和Mac OS X系统的Shell里已经包含，而Windows系统则需要PuTTY。由于我的本地计算机系统为 Arch Linux，下面则详细介绍Arch Linux下的步骤。
首先生成SSH的RSA公匙和私匙。在本地打开虚拟终端，输入：

cd ~/.ssh

如用户根目录下没有这个文件夹，则先新建一个：

mkdir ～/.ssh
cd ~/.ssh

然后生成秘匙对：

ssh-keygen -t rsa -C "your@email.com"

默认名称为id_rsa，如不修改则直接Enter，然后输入密码，两遍。

注意！！
输入密码的时候光标不会动，但系统内是输入进去了的，注意两遍密码一致。

添加SSH公匙

此时~/.ssh 目录下会出现 id_rsa id_rsa.pub两个文件。id_rsa是私匙，请不要泄露，妥善保存。在终端继续输入：

cat ~/.ssh/id_rsa.pub

将会出现一长串字符，结尾是你生成时输入的邮箱地址。将这噫长串字符全部复制下来，并粘贴到刚刚网页里填SSH pub Key的对话框内。然后确认。
等待几秒后Digital Ocean就能完成基础的配置了。

此时回到本地计算机的虚拟终端，输入：

ssh -t root@YOUR_vps_IP

其中root为用户名，@符号后面为你的vps的IP。没有意外应该就能连上远程服务器了。

之后…

到这里基本准备工作就差不多了。目前我们已经有了一个可以在本地通过SSH登录的VPS。之后的搭建下篇再说。
